SQL Task 1 — Price Changes
Write a SQL query to find products whose price changed from yesterday to today.
Expected output columns:
product_id, product_name, old_price, new_price


select product_id.today , product_name.today, price.yesturday as old_price, price.today as new_price
from products_today as today join products_yesturday as yesturday on product_id.today == product_id.yesturday where price.today<>price.yesturday;

SQL Task 2 — New Products
Write a SQL query to find products that are new today, meaning they exist in products_today but not in products_yesterday.
Expected output columns:
product_id, product_name, price, status

select product_id.today , product_name.today, price.today price, status.today, product_id.yesturday
from products_today as today left join products_yesturday as yesturday on product_id.today==product_id.yesturday where product_id.yesturday is null;

SQL Task 3 — Missing Products
Write a SQL query to find products that existed yesterday but are missing today.
Expected output columns:
product_id, product_name, price, status

select product_id.today , product_name.today, price.today price, status.today, product_id.yesturday
from products_today as today right join products_yesturday as yesturday on product_id.today==product_id.yesturday where product_id.today is null;

SQL Task 4 — Status Changes
Write a SQL query to find products whose status changed from yesterday to today.
Expected output columns:
product_id, product_name, old_status, new_status

select product_id.today , product_name.today, status.yesturday as old_status, status.today as new_status
from products_today as today join products_yesturday as yesturday on product_id.today == product_id.yesturday where status.today<>status.yesturday;
